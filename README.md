# Sales & Demand Forecasting for Businesses

## Objective

To forecast future sales using historical Superstore sales data.

## Dataset

Superstore Sales Dataset

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn
* Google Colab

## Project Steps

1. Data Cleaning
2. Feature Engineering
3. Monthly Sales Aggregation
4. Sales Trend Visualization
5. Linear Regression Forecasting
6. Model Evaluation using MAE

## Results

The model forecasts future sales trends based on historical business data.

## Business Insights

The forecast can help businesses:

* Plan inventory
* Manage staffing
* Improve budgeting
* Reduce overstocking and shortages
